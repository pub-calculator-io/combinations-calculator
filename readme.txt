Combinations Calculator calculates the number of ways of selecting r outcomes from n possibilities when the order of the items chosen in the subset does not matter.

[https://www.calculator.io/combinations-calculator/](https://www.calculator.io/combinations-calculator/)

== Usage ==

Add the Combinations Calculator shortcode to your page, post or sidebar:

`[ci_combinations_calculator]`

Add the following code to your template where you would like the Combinations Calculator to appear:

`<?php display_ci_combinations_calculator(); ?>`

1. Upload the Combinations Calculator /ci_combinations_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Combinations Calculator plugin through the "Plugins" menu in WordPress.
